Prueba Técnica - Desarrollo Front End con React
Este proyecto es una aplicación web interactiva desarrollada para la maquetación de un flujo de contratación de planes corporativos. Se enfoca en la fidelidad visual, la reutilización de componentes y una navegación fluida.

🚀 Tecnologías Utilizadas
React 18 - Librería principal.
Vite - Herramienta de compilación rápida.
React Router Dom v6 - Gestión de rutas y navegación.
CSS3 - Diseño modular y sistema de Grid/Flexbox.
Git - Control de versiones con historial semántico de commits.

📦 Estructura del Proyecto
/src/components: Componentes reutilizables (Button, Layout, PlanCard).
/src/pages: Pantallas individuales de cada paso del flujo.
/src/App.css: Estilos globales y diseño responsivo.

🛠️ Instalación y Ejecución

Sigue estos pasos para ejecutar el proyecto localmente:

1. Clonar el repositorio:
git clone https://github.com/Frank251090/prueba-tecnica-react.git

2.- Entrar a la carpeta:
cd frontend-app

3. Instalar dependencias:
npm install

4. Correr el servidor de desarrollo:
npm run dev

5. Abrir el navegador: Visita http://localhost:5173 para ver la aplicación.

💡 Notas de Implementación
Patrón Layout: Se utilizó un componente de envoltura para centralizar el panel lateral y el indicador de progreso.
Hooks: Uso de useState para la interactividad de selección de planes y opciones.
Navegación Dinámica: Flujo condicional entre Línea Nueva y Portabilidad.
